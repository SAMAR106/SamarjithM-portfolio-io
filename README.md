# Samarjith - AI/ML Developer Portfolio

A modern, interactive portfolio website showcasing expertise in AI/Machine Learning, Python development, Chatbot creation, and Web Development.

## Features

- **Hero Section**: Eye-catching introduction with animations
- **Skills Section**: Interactive skill cards with circular progress indicators
- **Projects**: Showcase of completed projects
- **About**: Professional background and experience
- **Contact**: Get in touch section
- **Responsive Design**: Fully responsive on all devices
- **Dark Mode Support**: Seamless light/dark theme switching

## Tech Stack

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **UI Components**: shadcn/ui
- **Testing**: Vitest + Playwright

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:8080](http://localhost:8080) in your browser.
